# Retro Games

A collection of self-contained retro arcade games built with vanilla HTML, CSS, and JavaScript. No dependencies, no build step — just open in a browser and play.

## Games

### Space Invaders (`spaceshooter.html`)
A Space Invaders-style arcade shooter. Defend against waves of descending aliens using pixel-perfect retro rendering on an HTML5 canvas.

- Arrow keys or WASD to move
- Spacebar to shoot

### Tic-Tac-Toe (`tictactoe.html`)
Classic Tic-Tac-Toe with a retro dark theme. Two players take turns on the same screen.

## How to Play

Just open any `.html` file directly in your browser — no server needed.

## Stack

- HTML5 Canvas (Space Shooter)
- Vanilla JavaScript
- Pure CSS styling
